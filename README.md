===========

Small pet-project service with [React](https://facebook.github.io/react/)

## Installation

- Install [node](http://nodejs.org/download/) or [n](https://github.com/tj/n), or [nvm](https://github.com/creationix/nvm)
- Install the dependencies

```bash
npm install
npm run dist
```

## Development
```
npm run watch
```
